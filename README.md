# AerialMapper

AerialMapper is a professional cross-platform drone photogrammetry application being developed from scratch.

## Vision

Develop a next-generation mapping software capable of generating:

- Orthomosaic
- DEM
- DSM
- DTM
- Point Cloud
- Mesh
- Textured Models
- GeoTIFF
- LAS/LAZ
- 3D Tiles

using AI-assisted processing, parallel computing, GPU acceleration, and a modern desktop interface.

---

## Features (Planned)

- Cross-platform (Windows & Linux)
- Electron + React Desktop GUI
- Python Photogrammetry Engine
- Future C++ Performance Modules
- Automatic Camera Calibration
- AI Image Quality Analysis
- Live Orthomosaic Preview
- Distributed Image Processing
- Multi-threaded Processing

- Project Management
